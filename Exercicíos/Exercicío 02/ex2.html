Tags!

Essas são as tags zikas que aprendi até agora

  • Br

    A TAG br serve para quebrar linhas;

  • h1 a h6

    AS TAGS de h1 até h6 servem para definir o tamanho da fonte dentro do código.

  • html

    TAG html serve para indicar o início e o fim do código.

  • Title

    A TAG title serve para mostrar o título

  • Styles

    A TAG styles serve para colocar cor dentro do código.

  • Background

    A TAG background serve para colocar uma cor no fundo no do código

  • Colour

    A TAG color serve pra dar cor nas letras

  • A TAG body é usada para definir o corpo do texto

  • Head

    A TAG head define a parte introdutória da página

  • Link

    A TAG link é usada para juntar um arquivo ao código;

  • img

    A TAG img é usada para introduzir uma foto ao código

  • Alt

    A TAG alt é uma tag que é usada para acessibilidade, quando um deficiente visual passa o cursor por cima da imagem ela é "falada"
